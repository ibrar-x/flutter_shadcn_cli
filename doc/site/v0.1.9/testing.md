# Testing

## Run Tests

```bash
cd shadcn_flutter_cli
flutter test
```

## Run Specific Tests

```bash
flutter test test/cli_integration_test.dart
```

## Notes

Tests use temporary directories and fixtures. Integration tests validate CLI install behavior.
