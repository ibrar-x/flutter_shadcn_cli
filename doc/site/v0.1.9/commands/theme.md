# theme (Legacy v0.1.9)

## Syntax

```bash
flutter_shadcn theme [--list | --apply <preset> | --apply-file <path> | --apply-url <url>]
```

## Behavior

- Applies theme presets/config to the single registry install target.
- No namespace-based theme routing.
