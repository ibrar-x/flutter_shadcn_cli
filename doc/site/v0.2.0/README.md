## Highlights

- Multi-registry resolution with default namespace
- Namespace addressing: `@namespace/component` and `namespace:component`
- `init <namespace>` inline init action execution from `registries.json`
- Optional file filters for add: `--include-files` / `--exclude-files`
- Registry commands: `registries`, `default`
