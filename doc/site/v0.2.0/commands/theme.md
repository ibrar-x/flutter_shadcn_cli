# theme

## Purpose
Apply a theme preset or custom theme JSON.

## Syntax

```bash
flutter_shadcn theme [--list|--apply <id>|--apply-file <path>|--apply-url <url>]
flutter_shadcn theme @<namespace> [--list|--apply <id>]
```

## Examples

```bash
flutter_shadcn theme --list
flutter_shadcn --experimental theme --apply-file theme.json
```
